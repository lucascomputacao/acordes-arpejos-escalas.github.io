Prova 1 — Tríades e inversões

Verifique se você domina a base harmônica antes de avançar para tétrades

Questão 1
Qual a fórmula intervalar de uma tríade maior?
A 1-2-5
B 1-3-5
C 1-♭3-5
D 1-3-♯5
Tríade maior: fundamental (1), terça maior (3) e quinta justa (5). A segunda opção está correta.
Questão 2
Em que inversão o acorde F/A está?
A Fundamental
B Primeira inversão (3ª no baixo)
C Segunda inversão (5ª no baixo)
D Terceira inversão
F/A tem a nota A (lá) no baixo. A é a terça de F (fá-lá-dó), logo está na primeira inversão.
Questão 3
Qual acorde do campo harmônico maior é diminuto?
A I
B ii
C V
D vii°
O sétimo grau (vii°) é o único acorde diminuto no campo harmônico maior. Ex: B° em C maior.
Questão 4
Qual a diferença entre tríade maior e menor?
A A quinta é diferente
B A terça é menor na tríade menor
C A fundamental muda
D Não há diferença
A tríade menor tem terça menor (♭3) em vez de terça maior. A quinta continua justa.
Questão 5
Quantas inversões possui uma tríade?
A 1
B 2
C 3
D 4
Tríade tem 3 notas, logo 2 inversões: 1ª (terça no baixo) e 2ª (quinta no baixo), além da posição fundamental.

Prova 2 — Tétrades e cifragem

Consolide acordes com sétima antes de partir para arpejos

Questão 1
Qual a diferença entre Cmaj7 e C7?
A Não há diferença
B Cmaj7 tem 7ª maior, C7 tem 7ª menor
C Cmaj7 é menor, C7 é maior
D C7 tem a quinta aumentada
Cmaj7 = C-E-G-B (7ª maior). C7 = C-E-G-B♭ (7ª menor). Essa é a diferença crucial.
Questão 2
Qual a função harmônica típica do acorde V7?
A Tônica
B Subdominante
C Dominante (tensão que resolve na tônica)
D Relativa menor
V7 é o acorde dominante. Ele cria tensão (pelo trítono entre 3ª e 7ª) e quer resolver no I.
Questão 3
Qual acorde aparece no 7º grau do campo harmônico maior?
A m7
B maj7
C 7
D m7♭5
Bm7♭5 em C maior. Também chamado de meio-diminuto (ø). Fórmula: 1-♭3-♭5-♭7.
Questão 4
O que significa o símbolo "ø" em um acorde?
A Diminuto
B Meio-diminuto (m7♭5)
C Aumentado
D Menor com 7ª maior
ø = meio-diminuto = m7♭5. Não confundir com ° (diminuto, que tem 7ª diminuta).
Questão 5
Qual o campo harmônico de C maior em tétrades?
A Cmaj7-Dm7-Em7-Fmaj7-G7-Am7-Bm7♭5
B C7-Dm7-Em7-F7-G7-Am7-Bm7
C Cmaj7-Dm7-E7-Fmaj7-Gm7-Am7-B7
D C-Dm-Em-F-G-Am-Bdim
Padrão: Imaj7 - iim7 - iiim7 - IVmaj7 - V7 - vim7 - viim7♭5. Decore isso!

Prova 3 — Arpejos e escalas básicas

Certifique-se de que consegue improvisar sobre acordes com arpejos e escalas

Questão 1
O que é um arpejo?
A Um acorde tocado com todas as notas ao mesmo tempo
B As notas de um acorde tocadas separadamente
C Uma escala tocada rapidamente
D Uma progressão de acordes
Arpejo = acorde quebrado. Você toca as notas separadas, em sequência, não simultaneamente.
Questão 2
Qual escala usar sobre um acorde Dm7 em contexto de ii-V-I em C maior?
A D menor harmônica
B D dórico (notas de C maior começando em D)
C D frígio
D D menor melódica
Sobre iim7, use dórico (relativo da escala maior da tonalidade). D dórico = notas de C maior.
Questão 3
Qual a fórmula da escala menor harmônica?
A T-sT-T-T-sT-T-T
B T-T-sT-T-T-T-sT
C T-sT-T-T-sT-T+sT-sT
D 1-2-♭3-4-5-♭6-7
Menor harmônica = menor natural com ♯7. Fórmula: 1-2-♭3-4-5-♭6-7. Intervalo de 1 tom e meio entre ♭6 e 7.
Questão 4
O que significa "superposição de arpejos"?
A Tocar dois arpejos ao mesmo tempo
B Tocar um arpejo diferente do acorde de base para criar tensões
C Tocar arpejos em oitavas diferentes
D Substituir acordes por arpejos
Ex: tocar arpejo de Em sobre Cmaj7 cria as tensões 9ª (E) e 13ª (A). É uma técnica de cor harmônica.
Questão 5
Quantas posições (digitações) da escala maior você deve conhecer?
A 1
B 3
C 5 ou 7
D 12
Padrão CAGED dá 5 posições. Sistema 3NPS dá 7. O importante é cobrir o braço inteiro.

Prova 4 — Modos e aplicação modal

Domine o pensamento modal antes de partir para a revisão final

Questão 1
O que é um modo?
A Uma escala diferente da maior
B Uma escala maior começando em graus diferentes
C Um tipo de acorde
D Uma técnica de improvisação
Modo = escala maior (ou outra escala-mãe) começando em outra nota. Mesmas notas, tônica diferente.
Questão 2
Qual modo usar sobre um acorde Dm7 em contexto modal (não ii-V-I)?
A D dórico
B D eólio
C D frígio
D Qualquer um, depende da vibe
Todos são menores. Dórico (6ª maior) é o mais comum, mas eólio e frígio também cabem. Contexto manda!
Questão 3
Qual o modo do 4º grau da escala maior?
A Jônico
B Dórico
C Lídio
D Mixolídio
Ordem: jônico (I), dórico (II), frígio (III), lídio (IV), mixolídio (V), eólio (VI), lócrio (VII).
Questão 4
Qual a nota característica do modo lídio?
A ♭3
B ♯4
C ♭7
D 7
Lídio = maior com ♯4. Som "flutuante", muito usado em acordes maj7 e composições modais.
Questão 5
Qual escala é o 7º modo da menor melódica?
A Lídio ♭7
B Alterada (super lócrio)
C Dórico ♭2
D Mixolídio ♭13
Escala alterada = 7º modo da menor melódica. Todas as tensões alteradas: ♭9, ♯9, ♯11, ♭13. Usada sobre dominantes alterados.

Questão 1
Qual a relação entre acordes, arpejos, escalas e modos?
A São conceitos independentes
B Acorde → arpejo (notas separadas) → escala (notas vizinhas) → modos (contexto)
C Modos geram acordes que geram arpejos
D Não há relação
Acorde = notas simultâneas. Arpejo = mesmas notas separadas. Escala = notas vizinhas que contêm o acorde. Modo = escala com contexto/tônica.
Questão 2
Como você escolheria a escala para improvisar sobre Am7-D7-Gmaj7?
A A eólio, D mixolídio, G jônico
B Escala de G maior em tudo
C A dórico, D lídio ♭7, G lídio
D Depende do contexto e da sonoridade que quero
Não há resposta única. Dórico/mixo/jônico é padrão, mas você pode alterar o D7 (alterada), usar lídio no G... contexto manda!
Questão 3
O que é mais importante: decorar padrões ou entender a lógica?
A Decorar padrões
B Entender a lógica
C Ambos — padrões dão fluência, lógica dá liberdade
D Nenhum — só tocar de ouvido
Decorar sem entender = tocar mecânico. Entender sem praticar = saber sem fazer. Precisa dos dois.
Questão 4
Você está pronto para livros mais avançados (Mick Goodrick, Jerry Bergonzi)?
A Sim, se domino este livro
B Não, preciso de mais repertório antes
C Sim, mas vou continuar revisando este
D Depende do meu objetivo musical
Se você fez o plano inteiro com consistência, pode avançar. Mas este livro é pra vida toda — sempre há o que revisar.
Questão 5
Qual o próximo passo depois de dominar acordes, arpejos, escalas e modos?
A Harmonia funcional avançada
B Improvisação sobre progressões complexas
C Transcrição de solos
D Todos os anteriores + tocar com outros músicos
Teoria sem prática é vazia. O próximo passo é USAR tudo isso: tocar com gente, transcrever, compor, improvisar. Música é verbo.
